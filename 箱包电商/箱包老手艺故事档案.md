# 箱包老手艺故事档案

白沟箱包老手艺-故事档案白沟箱包老手艺人：我给孙子做书包，他说「爷爷，你做的太慢了」一个关于手艺、孤独和「跟不上时代」的故事📦 大爷16岁学窝边，干了半辈子箱包「俺16岁学窝边，就是给箱包边包布条，得包的圆、包得紧。机器一快就跳线，全凭手稳。」他比划着捏针的姿势，眼里有光。最难的是上拉锁。他说：「箱包顺不顺滑，全看这道工序。尺儿对得准，布袋藏得巧，拉头上去『咔』一声脆响，这才叫活。」他拍拍胸脯：「俺做的，保你用十年不卡顿。」❤️ 那时候，老伴给俺打下手「夜里做到十一二点，她给俺煮挂面，卧个鸡蛋。」他声音低下去：「她走之前，给俺纳了半双鞋垫，说『有福啊，俺手不行了，你凑合穿』。」他摩挲着那半双鞋垫，眼眶红了。「俺学了半辈子没学会，她走了，这半双就当宝贝留着。」📱 手机嗡嗡响，他不敢接「老怕点错扣钱，村里老李家点错一下，没了200多。」他手忙脚乱划屏幕，儿子打视频，按半天没接上。「俺跟儿子视频，他老说『爸你照顾好自己』，俺就回『中中中』。其实俺想说——爹想你们啊。」话到嘴边，又咽成了回去。🧵 攒了30多年的线轴，画了七八遍的图纸他翻出铁盒，各色线轴摆一桌：「涤纶的、尼龙的、棉线的，配啥色俺都有。」又拿出卷边的图纸：「书包侧带要宽，能塞水杯。背带里垫海绵……」他声音发颤：「画了七八遍，眼睛花了，线都画不直。」😔 想给孙子做书包，怕人家嫌慢「俺不是非要做出个书包，就是想让娃背出去，跟同学吹一句——『我爷爷做的』。」他低头，声音小下去：「让他知道，俺不是只会吃饭等死的累赘。」「你看现在白沟的招工，全是油边工、裁料工，全自动裁床一按，几千件料子一下就出来了，谁还学俺这手工窝边、上拉锁？慢不说，还赚不了几个钱。可俺就是觉得，机器做的包，没魂。」村里会这老手艺的，走的走、忙的忙，就剩他一个了。「隔壁小媳妇会踩机子，可嫌俺慢，说『大爷您那套过时了』。」✨ 后来，他眼睛亮了我问：「要不我帮您留意，找个懂老手艺的人参谋参谋？」他愣住，红着眼忽然笑了：「真的？你别哄俺。」他把图纸叠得平平整整，压到枕头底下。「俺把料都拿出来，好的孬的，随便挑。」🌙 「现在夜里睡不着，俺就起来摸摸这些皮子，闻闻味儿，心里比吃安眠药还踏实。」他晃了晃手机：「这玩意儿老嗡嗡响，想得心慌，睡不着。」🚶 走的时候，他拽住我袖子「那啥……路上慢点。还有……谢谢啊。俺嘴笨，不会说漂亮话，就心里头热乎。」他站在院门口，一直望着，直到人影没了。📌 后记他说：「机器做的没魂。」我走远回头，他还扶着门框。手机又响了，这次他没慌。想起我说的「绿标志，不扣钱」，他轻轻戳了下去。视频，接通了。💬 你身边有这样守着老手艺的长辈吗？他们不是学不会新东西，是怕给孩子添麻烦；他们有满肚子的手艺，却没人愿意听。有空多跟家里老人唠唠，问问他们年轻时的得意事儿，他们真的会眼睛发亮。故事记录时间：2025年来源：白沟实地调研
